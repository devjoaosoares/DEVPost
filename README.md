# 🚀 DEVPost

O **DEVPost** é um protótipo de rede social nichada, desenvolvida para criar um ambiente de troca de informações, dicas e networking exclusivamente entre desenvolvedores.

Este projeto foi construído com o objetivo principal de praticar a integração de tecnologias Full Stack, desde a estruturação visual até a persistência de dados em um banco relacional.

---

## 🛠️ Tecnologias Utilizadas

O projeto utiliza uma stack sólida para o desenvolvimento web moderno:

* **Frontend:** HTML5, CSS3 e JavaScript (Vanilla).
* **Backend:** Node.js com o framework **Express**.
* **Banco de Dados:** **SQL** (PostgreSQL/MySQL) para gerenciamento de usuários e postagens.

---

## 📋 Funcionalidades Planejadas / Implementadas



- [ ] Cadastro e Login de desenvolvedores.
- [ ] Criação de postagens com suporte a blocos de código.
- [ ] Feed de notícias em tempo real.
- [ ] Perfil de usuário editável.
- [ ] Sistema de comentários e curtidas
